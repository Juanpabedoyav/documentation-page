Introducción

Visual Studio Code es un editor de código fuente desarrollado por Microsoft para Windows, Linux y macOS. Incluye soporte para la depuración, control integrado de Git, resaltado de sintaxis, finalización inteligente de código, fragmentos y refactorización de código. También es personalizable, por lo que los usuarios pueden cambiar el tema del editor, los atajos de teclado y las preferencias.


Chrome OS es un sistema operativo desarrollado por la compañía Google que se basa en el núcleo Linux y utiliza el navegador web Google Chrome como su principal interfaz de usuario, está basado en la nube. Por lo tanto, Chrome OS soporta principalmente aplicaciones web.

Configuración de espacio

Lo más importante para poder instalar Visual Studio Code en tu ChromeBook necesitas habilitar la versión de Linux (Beta) que viene por defecto en tu PC.

  • Abre las configuraciones de tu ChromeBook
  • En la barra lateral selecciona la opción de Linux(Beta)
  • Luego Encender
  • En el siguiente paso selecciona continuar
  • Elige tu usario y selecciona Instalar
Proceso de Instalación
Dirígete al Código de Visual Studio sitio web
Descargar el .debutante Paquete Debian
Abre el Archivos aplicación
Haga clic derecho y seleccione Instalar con Linux (Beta)

Desventajas

Pese a que sea un sistema bastante sencillo e intuitivo, es un sistema nuevo, a fin de cuentas.

Por eso, sabemos que habrá un considerable número de personas que por este hecho no querrán cambiar de aires.

Siempre es difícil entrar en un entorno que nunca has visitado y tener que acomodarte.

Un bajo rendimiento para tareas más pesadas

No hay soporte para algunas aplicaciones exclusivas de Windows/Mac

Beneficios

Otra interesante ventaja de Chrome OS y en esencia de los Chromebooks es que el almacenamiento de Google Drive nos llega integrado a la plataforma.

Con esto quiero decir que se accede a él como si de un "disco C:" se tratase. Pero más allá de las ventajas de su accesibilidad lo mejor es que esta allí.

Google nos regala 100 GB de almacenamiento gratuito en Google Drive durante dos años con la compra de un chromebook y con este tipo de almacenamiento integrado tenemos la opción de acceder a la información almacenada en Google Drive desde cualquier equipo, una ventaja inmensa que a día de hoy no está implantada de forma estandarizada en el resto de las plataformas como debería.

El precio es quizás el factor más atractivo de los Chromebooks y el que tira a más clientes en su dirección.
